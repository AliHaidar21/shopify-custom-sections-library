# Shopify Custom Sections Library

A portfolio collection of reusable Shopify theme sections built with Liquid, CSS, and Shopify schema settings.

This repo is for clean, reusable section code only not a full private Shopify theme.

1.Two option offer selector:
The Buy 2 button does not create a discount by itself. The store owner must first create an active Shopify discount code in Shopify Admin, then paste the exact same code into the section setting called “Offer discount code.” The section adds the configured quantity to cart and applies that code to the cart. Shopify decides whether the code is valid and applicable based on the discount rules created by the store owner.
